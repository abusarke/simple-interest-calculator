# Simple Interest Calculator

A shell script to calculate simple interest based on the principal, annual rate of interest, and time period in years.

## Instructions
This script is designed for educational purposes to demonstrate basic Bash scripting and Git version control.

## Input:
* principal: the amount of money borrowed or invested
* rate: the annual interest rate
* time: the time period in years

## Output:
* simple interest: the total interest calculated using (P * R * T) / 100
